# RPGInLetters

A text-based Role-Playing Game featuring turn-based combat and narrative choices. This project is designed as a hands-on study case for backend development concepts, OOP and even Game design! :D

## Learning Objectives

The primary goal of this project is to practice and demonstrate:

- Database Modeling.
- Entity Framework Core knowledge.
- Database Queries.
- Authentication (Implementing a basic user login/registration system).
- OOP Principles.

### Tech Stack

- Language: C# 14 / .NET 10.0
- ORM: Entity Framework Core
- Database: PostgreSQL
- Cloud/Hosting: Supabase

### Features Roadmap

- Turn-Based Combat
- Branching Narrative
- User System
- Persistent Data
- More...?
